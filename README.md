# The officiall site for our browser game - Shtrih The Cat And Friends

Final project for the course site engineering of the Digital Department of the Southern Federal University

##About

This project is a responsive React website created as a landing page for the browser fictional game **"Shtrih the cat and friends"**
The site includes:

* interactive section-based storytelling
* smooth navigation across different parts of the page
* typewriter dialogue effect
* responsive layout for desktop and mobile devices
* handmade pixel-art inspired visual style

##Technologies:
* React
* CSS
* JavaScript
* Vite
